Hypothesis ¶

We start by definining our null hypothesis as: tags and mentions have no impact on views for TikTok videos Our alternative hypothesis is that there is a correlation between tags and mentions and the number of views on tiktoks, that there is a significant difference from 0 for these variables.

Compiling Frequencies and Views ¶

Get Frequencies of Variables ¶

Now that we've wrangled the data and it is easier to work with, we can begin to analyze the correlation between mentions and playCount (views), and between hashtage and playCount. We can start by finding all hashtags used to see if any of them occur more than once, and doing the same with tags. We also have to take into account no tags and no mentions.

Connect Variables and Views ¶

As we can see from the hashtags, there is a strong sense of repition among them. However, for mentions, there is less repition. From this, I predict that the hashtags will have a stronger correlation to the number of views than the tags will. We can begin to graph these correlations, to see the total views per hashtag and per mention.

Is There a Linear Relationship? ¶

Now that we have proven some correlation between video views and hashtag frequency and video views and mention frequency, we can perform a linear regression to see if this is the truth. To do this, we need to have an array of the frequencies, the x values, and our views, our y values. This will tell us if we have a strong or loose correlation, and if it is positive or negative.

In [200]:

Based on our regression score, we can say that there exists a positive correlation between hashtag frequency and video views, and we can see that demonstrated in our regression line. We also have a p value of .003 for this variable, which is less than our significance level of .05, so we know that this is significant. In layman's terms, this means that overall, as the frequency of a hashtag increases, the number of views on the video also increases. We observe a t negative correlation between frequency of mentions and views in our regression line, but when observing the p-values, we have .845, which is greater than .05, so we can say that there is no statistical significance between mention frequency and views. So, we reject our null hypothesis for hashtags, but verify it for mentions.
